1. create
要素
create
覚えるときのイメージ
日本語でも使われるcreatorクリエイターやcreatureクリーチャーと関連させて覚える。
関連語
creator：クリエイター・創造主・作り手
creature：生き物・神の創造物
creation：創造
2. increase：増加する
223. decrease：減少する
783. concrete：具体的な
単語の歴史
ラテン語
creare（作る・育てる・生産する）→過去分詞creatusに由来する。
英語における変遷
15世紀初頭
生み出す・存在させるという意味で使われ、現代では「生み出す」が残っている。
補足
【create / cereal / increase / decrease の関係】
ラテン語creareから英語のcreateが生まれた。
このラテン語は、ローマ神話で農作物（特に穀物）の豊穣と収穫を司る神セレスCeresと関係がある。createの創り出すという意味と、Ceresの農作物を実らせるというイメージは似ている。
Ceresは穀物の豊穣と収穫を司る女神なので、英語のcereal「加工食品のシリアル・穀物」の語源となった。つまりcreateとcerealは語源的に関係がある。
またラテン語creareやCeresは、別のラテン語crescere「成長する」と関係があり、このcrescereはincrease, decreaseの語源である。
increaseは、in（中に）crescere（成長する）から構成され、膨らんで大きくなるイメージから「増加する」となる。
decreaseは、de（離れて）crescere（成長する）から構成され、成長から離れてしまうイメージから「減少する」となる。
2. increase
要素
in + crease
覚えるときのイメージ
もとになったラテン語からイメージをつかむ。
in（中に）crescere（成長する）から構成され、膨らんで大きくなるイメージから「増加する」となる。
関連語
223. decrease：減少する
783. concrete：具体的な
単語の歴史
ラテン語
in（中に）＋crescere（成長する）→increscere（増加する・成長する）
英語における変遷
14世紀半ば
増加する（自動詞）
14世紀後期
増やす（他動詞）
補足
【decreaseと関連させる】
de（離れて）crescere（成長する）から構成され、成長から離れてしまうイメージから「減少する」となる。increaseの対義語として覚えればよい。
【concludeと関連させる】
concreteは、con（共に）crescere（成長する）から構成され、共に成長するという意味からギュッと固まるイメージとなり「具体的な」となる。詳しい変遷は783. concreteへ。
3. improve
要素
im + pro + ve
覚えるときのイメージ
もとになった古フランス語やラテン語からイメージをつかむ。
en（〜させる）prode（利益）から構成され、利益にさせるという意味を経て「良くする・良くなる」となる。
en（〜させる）は、enjoy（楽しむ・楽しませる）・30. encourage（勇気を出させる・励ます）・716. endanger（危険にさらす）などにも見られる。
関連語
proud：誇りに思う
単語の歴史
ラテン語
pro（前に）＋esse（存在する）→prodesse（他のものより前に存在する→役に立つ）
→prode（有利な・利益をもたらす）
古フランス語
en（〜させる）＋prou（利益）＝利益にさせる
アングロ・フランス語
emprouwer（利益に変える）
英語における変遷
15世紀後期
利益に変える
1610年代
より良くする（他動詞）
1727年から
良くなる（自動詞）
補足
【proudとの関係】
prove部分の語源であるラテン語prode（有利な・利益をもたらす）は、後期古英語prud, prute「優れた・誇りに思う」となり、現代英語のproud「誇りに思う」となった。
4. mean
要素
mean
覚えるときのイメージ
meanは古い英語では「意図する」という意味だった。
mean to do（〜するつもり）にその意味が残っており、これが重要な表現。
「意図する」から言葉などで何かを伝えようと意図する→「意味する」となったと覚えればよい。

5. own
要素
own
覚えるときのイメージ
日本語でも使うowner（所有者・オーナー）からer（〜する人・〜するもの）を除いてown（所有する）となる。
関連語
737. owe：借りがある・〜を借りている・～のおかげである
ought：〜すべき（oweの過去形）
own（形容詞）：自身の
単語の歴史
古英語
agan（所有する）→geagnian
中英語
ounen（所有する）
英語における変遷
1200年頃
所有する
1300年頃
事実や罪を認める
1300年以降
この単語は使われなくなっていく
17世紀初期
名詞owner（所有者）から動詞ownが再び作られて使われるようになった。
補足
【own / oweはもともと同じ単語から生まれた】
ownとoweはどちらも古英語agan（所有する）から生まれた。
ownはそのまま「所有する」という意味で残った。
oweは古英語後期のagan to geldanne「支払うことを所有している」という表現から「借りがある」となった。
be owing to（～による・～のおかげである）は1650年代に使われ始めた表現。
6. include
要素
in + clude
覚えるときのイメージ
もとになったラテン語からイメージをつかむ。
in（中に）claudere（閉じる）から構成され、中に閉じ込めるというイメージから「〜を含む」となる。
cludeはclose（閉める）と関連し、どちらもラテン語claudere（閉じる）に由来する。
関連語
close：閉める
413. conclude：結論づける・締めくくる
1009. exclude：除外する・締め出す
単語の歴史
ラテン語
in（中に）＋claudere（閉じる）→inclaudere（閉じ込める・囲い込む）
英語における変遷
15世紀初頭
閉じ込める・含む
補足
【include / exclude / concludeの関係】
includeはin（中に）clude（閉じる）から中に閉じ込める→「を含む」となる。
excludeはex（外に）clude（閉じる）ので外に出して閉じる→「を除外する」となる。
concludeはcon（共に）clude（閉じる）ので共に閉じる→話を共に閉じる→「結論づける」となる。
7. consider
要素
con + sider
覚えるときのイメージ
もとになったラテン語からイメージをつかむ。
con（共に）sidus（天体・星）から構成され、星を観察する→よく見る→「考慮する・よく考える」となった。
consider O C（OをCと見なす）は「見なす」ではなく「OをCと考える」という訳でもよい。
関連語
239. desire：強く望む
単語の歴史
ラテン語
com（共に）＋sidus（天体・星）→considerare（星を観察する→よく見る）
古フランス語
considerer（熟考する・検討する）
中英語
consideren（熟考する・注意深く見る）
英語における変遷
14世紀後期
熟考する・注意深く見る
1530年代
特定の観点から見る・評価する
補足
【desireと関連させる】
desireはde（〜から）sidus（天体・星）から「星から」となる。
占星術に強い関心のあったローマ人がdesiderare「星がもたらすものを待ち望む」という意味で使った可能性があり、ここから「強く望む」となったと考えられる。
