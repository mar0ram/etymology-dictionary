1. create
要素
create
覚えるためのヒント
日本語でも使われるcreatorクリエイターやcreatureクリーチャーと関連させて覚える。
create(創造する, 引き起こす) creare（作る, 育てる, 引き起こす）というラテン語に由来する。
関連語
creator：クリエイター, 創造主, 作り手
creature：生き物, 神の創造物
creation：創造
cereal：穀物・穀物加工食品＝シリアル
【2.increase：増加する】 in（中に）crescere（成長する）というラテン語に由来する。 イメージは木の芽が膨らんで大きくなっていく感じ。 そこから「増加する」という意味になったと考える。
【223.decrease：減少する】 de（離れて）crescere（成長する）というラテン語に由来する。 成長から離れる→「減少する」となる。
【783.concrete：具体的な】 con（共に）crescere（成長する）というラテン語に由来する。 共に成長する→一体になる→ぎゅっと固まった感じ ここから「具体的な」となる。

【create / cereal / increase / decrease の関係】
createの語源はラテン語creare cerealの語源はラテン語Ceres increase, decreaseの語源はラテン語crescere これらのラテン語はすべて共通の語から生まれた兄弟語である。 3つとも作り出すこと、成長することに関連している。
【cerealの語源】
cerealは、ローマ神話で農作物（特に穀物）の豊穣と収穫を司る神セレスCeresに由来する。 cerealの意味はそのまま「穀物, 穀物加工食品シリアル」となっている。

8. allow
要素
al + low
覚えるためのヒント
allowは複雑な過程を経て「許す, 与える, 考慮する」という意味になったので歴史を知っておくと便利。
allowは al（=ad：～へ）locare（置く）からできたラテン語allocare（割り当てる, 与える）と al（=ad：～へ）laudare（称賛する）からできたラテン語allaudere（称賛する）が合体した単語。 allocareから「与える」、allaudereから「承認する, 許可する」となった。 また称賛する→評価する→評価の前段階として「考慮に入れる」という意味にもなった。 こうしてallowは「許す, 考慮に入れる, 与える」という意味を持つ。
関連語
【1823.allocate：割り当てる, 分配する】 allowの語源のひとつであるラテン語al（=ad：～へ）locare（置く）から生まれた。 allowとは兄弟関係にある単語。 ある場所へ置くイメージから「割り当てる, 分配する」というイメージ。
【417.locate：（受け身で）位置する】 ラテン語locare（置く）に由来する。
補足
【allowは異なる2つの単語の意味を引き継いだ】
ラテン語allocare（割り当てる, 与える）から「与える」という意味を引き継いだため、名詞allowanceには与えるお金＝「手当, 小遣い」の意味がある。
またラテン語allaudare（称賛する, 承認する）から「許す」という意味を引き継いだため、名詞allowanceには「許可」の意味もある。
【allowのal】

15. store
要素
store
覚えるためのヒント
動詞storeは名詞と関連させて覚える。
名詞storeを見ると「店」という意味が最初に浮かぶかもしれない。 しかしstoreはもともと「家庭内の備蓄, 保管」を意味した。 そこから家庭内に限らず一般的な「備蓄, 保管」となる。 さらに「備蓄, 保管を置く場所」つまり「店」という意味が生まれた。
つまりstoreのもともとの意味は「備蓄, 保管」であり、動詞store「保存する, 保管する, 蓄える」はその意味を残している。
補足
【ターゲット1000：943.in store (for)】 これはもともと「将来使うために備蓄してある」という意味。 ここから比喩的に出来事が「降りかかろうとしている」「用意されている」となった。

tendは tendere（伸ばす, 引っ張る）というラテン語に由来する。 心や注意を対象に伸ばす→対象の方へ「傾く」というイメージ。
関連語
【205.attend：世話をする, 対処する, 出席する】 at（=ad：〜へ）tendere（伸ばす）というラテン語に由来する。 〜へ伸ばす →心や労力を相手へ伸ばす→「世話をする」 →労力を問題へ伸ばす→「対処する」 →イベントへ(足を)伸ばす→「出席する」
【1732.contend：主張する, 競う】 con（共に）tendere（伸ばす）というラテン語に由来する。 相手と共に伸ばす→「競う, 対立する」→対立の中で「主張する」
【339.extend：伸ばす, 延ばす】 ex（外に）tendere（伸ばす）というラテン語に由来する。 外に伸ばす→「伸ばす, 延ばす」
【432.intend：意図する】 in（中に）tendere（伸ばす）というラテン語に由来する。 注意を中に伸ばす→「意図する」
【1187.tense：張り詰めた, 緊張した】 tendere（伸ばす）というラテン語の過去分詞tensusに由来する。 ピンと伸ばされた→弦を引っ張るイメージから「緊張した, 張り詰めた」となる。
補足
【ターゲット1000：745.tend to do】 to do（～すること）の方にtend（傾く） ここから「～する傾向がある」となる。
17. concern
要素

覚えるためのヒント
もとになったラテン語からイメージをつかむ。

関連語
【85.certain：確信して, 確かな, ある～】 cernere（ふるいにかける）というラテン語に由来する。 ふるいにかけると識別される→確定している状態となる。 そこから事実と照らして確定している＝「確信している, 確かな」となった。 また事実は確定しているが、読み手や聞き手はその情報を知らないという場合にa certain　～「ある～」となった。
【865.discrimination：差別・区別】 dis（離れて）cernere（ふるいにかける）というラテン語に由来する。 ふるいにかけて離すイメージから「区別, 差別」

【ターゲット1000：925.ascribe：AをBのせいにする】 ac（=ad：〜へ）scribere（書く）というラテン語に由来する。 〜へ書く→リストや文書へ書いて追加するイメージ 例えば所持品と持ち主のリストを作って物と人を結び付ける感じ。 ここからAをBと結び付ける→「AをBのせいにする」となる。
