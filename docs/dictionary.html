1. create
要素
create
関連語
creature「生き物」
creation「創造」
単語の歴史
ラテン語
creare「作る・生産する」
過去分詞
creatus
英語での意味の変遷
15世紀初頭：「生み出す・存在させる」
補足
ラテン語creareは、ローマ神話の豊穣の女神セレスCeres＝農作物（特に穀物）の収穫を司る神に関連する。
Ceresはcereal「加工食品のシリアル・穀物」の語源。
またCeresは、ラテン語crescere「成長する」と関連する。
crescereはincrease, decreaseの語源。
increase：in「中に」＋crescere「成長する」→膨らんで大きくなるイメージ→「成長する」
decrease：de「離れて」＋crescere「成長する」→成長の程度が減るイメージ→「減少する」
2. increase
要素
in + crease
覚えるときのイメージ
increase：in「中に」＋crescere「成長する」→膨らんで大きくなるイメージ→「成長する」
単語の歴史
ラテン語
in「中に」＋crescere「成長する」
increscere「増加する」
英語での意味の変遷
14世紀半ば：（自動詞）「増加する」
14世紀後期：他動詞「増やす」
3. improve
要素
im + pro + ve
覚えるときのイメージ
in+prove：en「〜させる」＋prode「利益」→「利益にさせる」→「良くする」「良くなる」
関連語
proud「誇りに思う」
単語の歴史
ラテン語
pro「前に」＋esse「存在する」
prodesse「前に存在する・役に立つ」
prode「有利な・利益をもたらす」
古フランス語
en「〜させる」＋prou「利益」＝「利益にさせる」
アングロ・フランス語
emprouwer「利益に変える」
英語での意味の変遷
15世紀後期：「利益に変える」
1610年代：他動詞「より良くする」
1727年から：自動詞「良くなる」
補足
prove部分の語源となるラテン語prodeは、後期古英語prud, prute「優れた・プライドを持つ・誇りに思う」となり、現代英語のproud「誇りに思う」となった。
4. mean
要素
mean
覚えるときのイメージ
mean：もともとは「意図する」と言う意味。
mean to do「〜するつもり」。
「意図する」から「意味する」という意味にもなった。
単語の歴史
古英語
mænan「何かをしようと意図する・ある言葉を伝える」
中英語
mēnen「意図する・心に抱く」
英語での意味の変遷
14世紀後期：「特定の種類の意図を持つ」
1888年までに：「重要である・意味を持つ」
5. own
要素
own
覚えるときのイメージ
owner「所有者・オーナー」から-er「〜する人・〜するもの」を取り除いてown「所有する」
関連語
owe「借りがある・－を借りている・～のおかげである」
ought：oweの過去形で「－すべきである」
単語の歴史
古英語
agan「所有する」
geagnian
中英語
ounen「所有する」
英語での意味の変遷
1200年頃：「所有する」
1300年頃：この語は使われなくなる
17世紀初期：名詞owner「所有者」から動詞ownが再び作られ、使われるようになった。
補足
ownとoweはどちらも古英語agan「所有する」に由来する。
own：そのまま「所有する」という意味で残った。
owe：agan to geldanne「支払うことを所有している」という表現から「借りがある」となった。
be owing to：1650年代be owing to「～による・～のおかげである」が使われ始めた。
